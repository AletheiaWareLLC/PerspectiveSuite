# PerspectiveSuite

Clone using --recursive:
```
$ git clone --resursive ....
```

If you have cloned without --recursive then:
```
$ git submodule update --init --recursive
```

Builds Joy, Perspective, AletheiaWareCommonJava, JoyJava and PerspectiveJava
